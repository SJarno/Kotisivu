Hieman itsestäni

Koodaajan alku Helsingistä, joka opiskelee avoimessa yliopistossa tietojenkäsittelytieteitä. Teen tällä hetkellä lähinnä sivuprojekteina webbidevausta ja erilaisia käyttäjäsovelluksia. Samalla opetellen uusia ohjelmointitaitoja milloin mistäkin lähteestä.

Haaveena ja tavoitteena on tehdä työtä alalla tavalla tai toisella tulevaisuudessa.

Taidot

Mitä ohjelmointitaitoihin tulee, omaan kokemusta muun muassa Javasta, mistä minulla on eniten kokemusta tällä hetkellä. Mukaan matkaan on luonnollisesti tarttunut muitakin työkaluja, kuten perus webbidevaukseen liittyvät HTML, CSS ja JavaScript. Myös relaatitietokannat ja niiden parissa puuhastelu on tullut tutuksi jonkin verran. Pintaraapaisua löytyy palvelinohjelmoinnista Javalla jonkin verran.

Lyhyenä listana tutut ohjelmointikielet menee seuraavanlaisesti

  • Html
  • CSS
  • JavaScript
  • Java
  • Sql

Työkalut ja muut teknologiat

Muutamia kirjastoja ja muita tuttuja teknologioita on muun muassa seuraavia:

  • Netbeans (Java)
  • Maven
  • H2-tietokanta (Java)
  • SQLite
  • Visual Studio Code
  • Ubuntu
  • Git ja GitHub

Koulutus

Kuten aiemmin mainittu, opiskelen tällä hetkellä Helsingin avoimessa yliopistossa ensimmäisen vuoden opintoja tietojenkäsittelytieteellisessä. Tavoitteena on samalla oppia uusia taitoja niin paljon kuin pystyn ja saada opiskelupaikka lähitulevaisuudessa.

Käytyjä kurseja löytyy seuraavanlaisesti tällä hetkellä:

  • Ohjelmoinnin perusteet Java
  • Ohjelmoinnin jatkokurssi Java
  • Tietokantojen perusteet(va. 2020)
  • Tietokone työvälineenä
  • Fullstack Webohjelmointi(va. 2020)
